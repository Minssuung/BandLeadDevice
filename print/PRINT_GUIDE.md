## 3. 조립 순서 (PCB 중심)

**그립이 좌우 클램쉘** — 열린 반쪽에 부품 눕히고 닫아 나사. 조립 쉬움.
1. **각 반쪽에 부품** (열린 상태라 접근 쉬움)
   - **왼쪽**: IMU → -X벽 플레이트, MAX485 → 옆 / AH49E → 앞 홀포켓 / 트리거 보스(좌)
   - **오른쪽**: SS-5GL → 측면 포켓+나사2(피치6.4), 레버 측면창 / 트리거 보스(우)
   - 트리거 레버: 자석 Ø5×2 끼우고, 양쪽 보스 사이 + Φ3 핀, 토션스프링(0.4×ID3.2)
2. **허브보드 + 배선** — 허브보드를 하부 헤드 트레이에 놓고, 모든 선(조이스틱5·홀3·리프트2·버튼·IMU→MAX485) 허브보드에 직배선 → **바닥 Ø8 출구로 외부 케이블 1개**
3. **좌우 반쪽 닫고 시임 나사 3개** 조임 (헤드 뒤·핸들 2)
4. **캐리어②에 조이스틱·버튼 장착**: KY-023 → 앞 4스탠드오프(29.1×23.4), 택트2 → 홀더에 밑서 삽입
5. **캐리어를 그립 헤드에 스냅** (5훅) → **버튼캡 2개** 끼움

## 4. 손가락 매핑 (작동)

- **엄지**: 조이스틱(주행) + A/B/menu 버튼캡(토크OFF·키보드 등)
- **검지**: 트리거(그리퍼) — 당기면 자석이 AH49E에 1~3mm로 접근해 아날로그 센싱
- **중지**: 측면 리프트 SS-5GL(자체 클릭)
- **내부**: IMU(손목/그립 자세, 18° 세로)

## 6. 미관 마감 (프린트 검증 후)

기능이 실물로 확정되면 **Blender**로 grip_body 외형만 Smooth/Sculpt (내부 정밀 유지). cadquery 파라메트릭으론 박스-원통 경계 블렌딩이 안 됨(fillet·loft 실패 확인). 메시 스무딩 목표 형상은 `cad/out/controller_smooth.stl` 참고.
