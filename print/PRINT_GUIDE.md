## 3. 조립 순서 (PCB 중심)

1. **캐리어에 조이스틱·보드 장착**
   - KY-023 모듈 → 캐리어 앞 4 스탠드오프(27×20)에 M2, 스틱은 돔홀로 (모듈 5핀은 선만 빼둠)
   - 택트 2개 → 만능보드(40×18)에 납땜(액추에이터 위로) → 보드를 캐리어 뒤 스탠드오프 4에 M2 (택트가 버튼홀에 정렬)
2. **그립 본체①에 부품 장착**
   - IMU → -X벽 M2 스탠드오프 2 (대각 홀) / MAX485 → 그 옆 +X cavity에 글루/타이
   - SS-5GL → 측면 포켓 M2 ×2, 레버가 측면 창으로(중지)
   - AH49E → 앞쪽 홀 마운트바 포켓(자석 마주봄)
   - 트리거: 레버에 자석 Ø5×2 → 피벗보스 사이에 → Φ3 핀 관통 → 토션스프링
3. **직배선** — 조이스틱5·홀3·리프트2 → 보드 junction / IMU→MAX485→보드 / 보드→외부 LOLIN D32 케이블은 **하단 Ø8 출구**로
4. **캐리어(+조이스틱+보드)를 그립 헤드에 스냅** — 5훅(앞2+측2+뒤1)이 림 창에 딸깍
5. **버튼캡 2개를 캐리어 구멍에 끼움** — 디스크가 위로 못 빠지고, 아래 택트가 받쳐 눌림

## 4. 손가락 매핑 (작동)

- **엄지**: 조이스틱(주행) + A/B/menu 버튼캡(토크OFF·키보드 등)
- **검지**: 트리거(그리퍼) — 당기면 자석이 AH49E에 1~3mm로 접근해 아날로그 센싱
- **중지**: 측면 리프트 SS-5GL(자체 클릭)
- **내부**: IMU(손목/그립 자세, 18° 세로)

## 6. 미관 마감 (프린트 검증 후)

기능이 실물로 확정되면 **Blender**로 grip_body 외형만 Smooth/Sculpt (내부 정밀 유지). cadquery 파라메트릭으론 박스-원통 경계 블렌딩이 안 됨(fillet·loft 실패 확인). 메시 스무딩 목표 형상은 `cad/out/controller_smooth.stl` 참고.
