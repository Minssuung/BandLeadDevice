## 3. 조립 순서 (PCB 중심)

**그립이 좌우 클램쉘** — 열린 반쪽에 부품 눕히고 닫아 나사. 조립 쉬움.
1. **각 반쪽에 부품** (열린 상태라 접근 쉬움)
   - **왼쪽**: IMU → -X벽 플레이트, MAX485 → 옆 / AH49E → 앞 홀포켓 / 트리거·리프트 보스(좌)
   - **오른쪽**: 트리거·리프트 보스(우)
   - **SS-5GL(중지 리미트) → 손잡이 안 +X보스에 2나사 셀프탭**(액추에이터 -Y=레버쪽). 2선 직배선
   - 검지·중지 트리거 레버: 양쪽보스 Φ3핀 + **토션스프링**(코일→핀, 다리 양끝 관통홀에 끼워 끝 꺾어 걺). 검지=자석 Ø5×2(홀센서가 읽음) / 중지=날 뒷면이 **SS-5GL 누름**(Meta Quest 그립버튼식)
2. **허브보드 + 배선** — 허브보드를 하부 헤드 트레이에 놓고, 모든 선(조이스틱5·홀3·리프트2·버튼·IMU→MAX485) 허브보드에 직배선 → **바닥 Ø8 출구로 외부 케이블 1개**
3. **좌우 반쪽 닫고 시임 나사 4개**(M2×20) — **왼쪽 옆면 접근홀로** 드라이버 넣어 조임(헤드뒤·정크션뒤·핸들하부 앞뒤)
4. **캐리어②에 조이스틱·버튼 장착**: KY-023 → 앞 4스탠드오프(29.1×23.4), 택트2 → 홀더에 밑서 삽입
5. **캐리어를 그립 헤드에 스냅** (5훅) → **버튼캡 2개** 끼움

## 4. 손가락 매핑 (작동)

- **엄지**: 조이스틱(주행) + A/B/menu 버튼캡(토크OFF·키보드 등)
- **검지**: 트리거(그리퍼) — 당기면 자석이 AH49E에 1~3mm로 접근해 아날로그 센싱
- **중지**: 앞면 2단 트리거 — 당기면 레버 날 뒷면이 **SS-5GL** 누름 + 토션스프링 복귀 (Meta Quest 그립버튼식)
- **내부**: IMU(손목/그립 자세, 18° 세로)

## 6. 미관 마감 (프린트 검증 후)

기능이 실물로 확정되면 **Blender**로 grip_body 외형만 Smooth/Sculpt (내부 정밀 유지).

**미관 타깃 = 실제 Meta Quest 3 컨트롤러 스캔** (`cad/ref/quest3_full_right.stl` = 완전체, `cad/ref/quest3_cover_right.stl` = 손잡이 겉쉘). 곡선 에르고 손잡이 + 라운드 헤드 + 유기적 블렌딩.

**Blender 워크플로**:
1. 우리 `grip_body_v3.stl`(또는 좌우 합본)을 Quest 스캔과 정렬(헤드·손잡이축 맞춤)
2. 겉면을 Quest 스캔 형상에 맞춰 조각(Shrinkwrap/Sculpt) — **내부 마운트·포켓·홀은 유지**
3. ⚠️ **손잡이 단면은 우리 치수 유지**: IMU(폭30)+보드 때문에 Φ38 필요 → Quest(단면 27×28)보다 두꺼움. **스타일(곡률·라운드)만 맞추고 굵기는 우리 것**

> cadquery 파라메트릭으론 박스-원통 경계 블렌딩이 안 됨(fillet·loft 실패 확인) → 이 단계는 메시 기반(Blender)이 정석.
