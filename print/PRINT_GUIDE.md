# Bandi V2 리드디바이스 — 프린트 & 조립 가이드 (턴키)

V3 파라메트릭 컨트롤러. **기능 검증 완료**(2라운드 독립 fact-check 통과 — 스냅 잠김·간섭·셸내부·watertight 전부 OK). 외형 미관 마감은 프린트 검증 후 별도(Blender 권장).

> ⚠️ 이건 **기능 프로토타입** 형상이라 약간 각져. 손에 쥐어 핏/작동 확인용. 매끈한 에르고 마감은 프린트로 기능 확정 후.

## 3. 조립 순서

1. **그립 본체①에 부품 장착**
   - IMU → -X벽 M2 스탠드오프 2개에 나사 고정 (대각 홀)
   - SS-5GL → 측면 포켓에 넣고 M2 ×2, 금속레버가 측면 창으로 나옴(중지)
   - AH49E → 앞쪽 홀 마운트바 포켓에 삽입
   - 트리거: 레버 포켓에 자석 Ø5×2 끼움 → 레버를 피벗보스 사이에 → Φ3 핀 관통 → 토션스프링(핀에, 한 다리는 그립 레그포스트, 한 다리는 레버)
   - 모든 배선을 **하단 Ø8 배선출구**로 빼서 외부 MCU로
2. **헤드 캐리어②에 부품 장착**
   - KY-023 → 4보스에 M2 (스틱 Ø18 돔홀로 돌출)
   - 택트 3개 → 밑면 홀더 3에 삽입(액추에이터 위로)
3. **캐리어를 그립 헤드에 스냅** — 5개 캔틸레버 훅이 그립 림 창에 딸깍 (앞 2 + 측 2 + 뒤 1)
4. **버튼캡 3개를 캐리어 구멍에 끼움** — 슬릿 플랜지가 압축돼 통과 후 빠짐방지

---

## 4. 손가락 매핑 (작동)

- **엄지**: 조이스틱(주행) + A/B/menu 버튼캡(토크OFF·키보드 등)
- **검지**: 트리거(그리퍼) — 당기면 자석이 AH49E에 1~3mm로 접근해 아날로그 센싱
- **중지**: 측면 리프트 SS-5GL(자체 클릭)
- **내부**: IMU(손목/그립 자세, 18° 세로)

## 5. 핏 안 맞으면 (전부 파라메트릭, 즉시 재생성)

| 증상 | 조정 (`cad/parts.py` 또는 스크립트) |
|---|---|
| 스냅 헐렁/빡빡 | `SNAP_CLEAR`(0.15) / 훅 `LIP_OUT` |
| 부품 포켓 빡빡 | `FDM_CLEAR`(0.3) |
| 버튼캡 안 들어감(PLA 뻣뻣) | PETG로 출력 / 슬릿 폭↑ |
| 트리거 센싱 약함 | 자석-홀 갭(`HALL`/`MAG` 위치) |
| 트리거 끝에서 빡빡 | 14°에서 멈추거나 슬롯 모서리 더 깎기 |

재생성: `cad/.venv/bin/python cad/grip_body_v3.py && .../assemble_v3.py` → `cad/.venv/bin/python cad/audit_v3.py`로 자가검증.

## 6. 미관 마감 (프린트 검증 후)

기능이 실물로 확정되면 **Blender**로 grip_body 외형만 Smooth/Sculpt (내부 정밀 유지). cadquery 파라메트릭으론 박스-원통 경계 블렌딩이 안 됨(fillet·loft 실패 확인). 메시 스무딩 목표 형상은 `cad/out/controller_smooth.stl` 참고.
