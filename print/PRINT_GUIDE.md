## 3. 조립 순서 (PCB 중심)

1. **만능보드(40×44)에 실장 + 직배선**
   - 조이스틱 KY-023 → 보드 4홀(27×20)에 M2, 스틱 위로
   - 택트 3개 → 보드에 납땜(액추에이터 위로, 캐리어 버튼홀에 정렬)
   - (옵션) MAX485·분압 등도 이 보드 위 또는 옆
2. **보드를 캐리어 밑면 스탠드오프 4개에 M2 고정** — 스틱은 돔홀, 택트는 버튼홀에 정렬
3. **그립 본체①에 부품 장착**
   - IMU → -X벽 M2 스탠드오프 2 (대각 홀)
   - SS-5GL → 측면 포켓 M2 ×2, 레버가 측면 창으로(중지)
   - AH49E → 앞쪽 홀 마운트바 포켓
   - 트리거: 레버에 자석 Ø5×2 → 피벗보스 사이에 → Φ3 핀 관통 → 토션스프링
4. **직배선 연결** — 그립측(홀·리프트·IMU) ↔ 보드(또는 외부 MCU), 보드↔외부 LOLIN D32 케이블은 **하단 Ø8 배선출구**로
5. **캐리어(+보드)를 그립 헤드에 스냅** — 5훅(앞2+측2+뒤1)이 림 창에 딸깍
6. **버튼캡 3개를 캐리어 구멍에 끼움** — 디스크가 위로 못 빠지고, 아래 택트가 받쳐 눌림(플랜지 불필요)

---

## 4. 손가락 매핑 (작동)

- **엄지**: 조이스틱(주행) + A/B/menu 버튼캡(토크OFF·키보드 등)
- **검지**: 트리거(그리퍼) — 당기면 자석이 AH49E에 1~3mm로 접근해 아날로그 센싱
- **중지**: 측면 리프트 SS-5GL(자체 클릭)
- **내부**: IMU(손목/그립 자세, 18° 세로)

## 6. 미관 마감 (프린트 검증 후)

기능이 실물로 확정되면 **Blender**로 grip_body 외형만 Smooth/Sculpt (내부 정밀 유지). cadquery 파라메트릭으론 박스-원통 경계 블렌딩이 안 됨(fillet·loft 실패 확인). 메시 스무딩 목표 형상은 `cad/out/controller_smooth.stl` 참고.
