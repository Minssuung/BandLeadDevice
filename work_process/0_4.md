# 0_4. 로봇팔 관절 매핑표

> **OpenArm J1~J8 관절과 IMU 축 대응 — 이미 정리 완료**
> 상세 내용은 [1_1.md](./1_1.md) 참고

---

## 매핑 요약

| IMU 위치 | 제어 관절 | 인체 동작 |
|----------|----------|---------|
| 윗팔 밴드 (이두근) | J1, J2, J3 | 어깨 3 DOF (Yaw/Pitch/Roll) |
| 아랫팔 밴드 (전완근) | J4 | 팔꿈치 굴곡/신전 |
| 손잡이 IMU | J5, J6, J7 | 손목 3 DOF (Yaw/Roll/Pitch) |
| 버튼 1 | J8 반시계(CCW) | 그리퍼 닫기 (집기) |
| 버튼 2 | J8 시계(CW) | 그리퍼 열기 (펴기) |

---

## 이 단계에서 파악해야 할 핵심

- OpenArm은 **7 DOF + 그리퍼(J8) 1 DOF** 구성
- IMU 3개로 7개 관절을 담당:
  - 윗팔 1개 → J1, J2, J3 (어깨 3축)
  - 아랫팔 1개 → J4 (팔꿈치 1축)
  - 손잡이 1개 → J5, J6, J7 (손목 3축)
- 버튼 2개 → J8 그리퍼 제어 (IMU 불필요)
