# 로봇팔 관절 매핑표 (IMU 축 → OpenArm 관절 각도 대응)

> 대상 로봇: **OpenArm v1.0** (Enactic, 7 DOF + Gripper 1 DOF)
> 참고: [openarm.dev](https://openarm.dev) / [docs.openarm.dev](https://docs.openarm.dev) / [URDF](https://github.com/enactic/openarm_description)

## 1. OpenArm 관절 구조 요약

URDF(`openarm_arm.xacro`)에서 확인된 각 관절의 회전축 및 가동 범위:

| 관절 | URDF 회전축 | 가동 범위 (rad) | 가동 범위 (도) | 대응 인체 동작 |
|------|------------|----------------|--------------|--------------|
| J1 | Z축 (0,0,1) | −1.40 ~ +3.49 | −80° ~ +200° | 어깨 수평 회전 (Yaw, 내회전/외회전) |
| J2 | −X축 (−1,0,0) | −1.75 ~ +1.75 | −100° ~ +100° | 어깨 굴곡/신전 (Pitch, 팔 올리기/내리기) |
| J3 | Z축 (0,0,1) | −1.57 ~ +1.57 | −90° ~ +90° | 상완 축 회전 (Roll, 내회전/외회전 twist) |
| J4 | Y축 (0,1,0) | 0.00 ~ +2.44 | 0° ~ +140° | 팔꿈치 굴곡/신전 (Elbow flex/extend) |
| J5 | Z축 (0,0,1) | −1.57 ~ +1.57 | −90° ~ +90° | 손목 Yaw / 전완 회내·회외 |
| J6 | X축 (1,0,0) | −0.79 ~ +0.79 | −45° ~ +45° | 손목 Roll (요측/척측 편위) |
| J7 | Y축 (0,±1,0) | −1.57 ~ +1.57 | −90° ~ +90° | 손목 Pitch (굴곡/신전) |
| J8 | (그리퍼 모터) | 0 ~ 1.05 rad | 0° ~ 60° | 그리퍼 열기/닫기 |

> J8 기준: 0° = 완전 닫힘(집기), **+60° 시계 방향 = 완전 열림(펴기)**
> (출처: [OpenArm Gripper Spec](https://docs.openarm.dev/hardware/specifications/gripper))

## 3. IMU 축 → 관절 각도 세부 대응

### 3-1. 윗팔 IMU (J1, J2, J3) — 어깨 3 DOF

상완 IMU의 절대 자세(쿼터니언)에서 Euler 분해:

| IMU Euler 성분 | 대응 OpenArm 관절 | 물리적 의미 |
|---------------|-----------------|------------|
| Yaw (Z축 회전) | J1 | 어깨 수평 회전 (팔을 앞/뒤로 swing) |
| Pitch (Y축 회전) | J2 | 어깨 굴곡/신전 (팔을 위/아래로 올림) |
| Roll (X축 회전) | J3 | 상완 축 비틀기 (내회전/외회전) |

> **주의:** 착용 방향에 따라 부호(±)가 반전될 수 있으므로, 캘리브레이션 단계에서 각 축 부호를 실측 후 확정해야 함.

### 3-2. 아랫팔 IMU (J4) — 팔꿈치 1 DOF

- 상완 IMU와 전완 IMU의 **상대 쿼터니언 차이**에서 Y축 성분 추출 → J4 각도
- J4 범위: 0° (팔 펼침) ~ 140° (완전 굽힘)

| 동작 | J4 값 |
|------|-------|
| 팔 완전 펼침 | 0 rad (0°) |
| 팔 완전 굽힘 | 2.44 rad (~140°) |

### 3-3. 손잡이 IMU (J5, J6, J7) — 손목 3 DOF

전완 IMU 기준, 손잡이 IMU의 **상대 자세**에서 Euler 분해:

| IMU Euler 성분 | 대응 OpenArm 관절 | 물리적 의미 |
|---------------|-----------------|------------|
| Yaw (Z축 회전) | J5 | 손목 Yaw / 전완 회내·회외 (±90°) |
| Roll (X축 회전) | J6 | 손목 요측·척측 편위 (±45°) |
| Pitch (Y축 회전) | J7 | 손목 굴곡/신전 (±90°) |

### 3-4. 버튼 → J8 그리퍼 제어

| 버튼 | J8 회전 방향 | 동작 | OpenArm 기준 |
|------|------------|------|-------------|
| 버튼 1 | **반시계(CCW)** | 집기 (닫기) | 0° 방향 (zero position = 완전 닫힘) |
| 버튼 2 | **시계(CW)** | 펴기 (열기) | +60° (완전 열림) |

> ⚠️ **주의 (수정 필요):**
> 최초 설계에서 버튼 방향이 **반대**였으므로 수정함.
> OpenArm 공식 스펙: *"시계 방향 +60° = 완전 열림(펴기)"*
> → 버튼 1(집기) = **반시계**, 버튼 2(펴기) = **시계** 로 수정.

## 4. 전체 Data Flow 개요

```
[윗팔 IMU] ──> 절대 Quaternion ──> Euler(Yaw/Pitch/Roll) ──> J1, J2, J3
[전완 IMU] ──> 상대 Quaternion (vs 윗팔) ──> Y축 성분 ──> J4
[손잡이 IMU] ──> 상대 Quaternion (vs 전완) ──> Euler(Yaw/Roll/Pitch) ──> J5, J6, J7
[버튼 1 (집기)] ──> J8 CCW 명령
[버튼 2 (펴기)] ──> J8 CW 명령
```

## 5. 향후 확인 사항

- [ ] 각 IMU의 착용 기준 좌표계 정의 (Zero-pose 캘리브레이션 자세)
- [ ] J1~J7 관절 한계각 클리핑 로직 구현
- [ ] 각 관절 부호(±) 실측 검증 (사람 동작 방향 ↔ 로봇 방향 일치 여부)
- [ ] J4 오프셋 보정 (팔꿈치 완전 펼침 시 = 0 기준 정렬)
- [ ] 그리퍼 버튼 토글 방식 확정 (누르는 동안 유지 vs 한 번 누르면 toggle)
